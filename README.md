## What this is

A single-page personal site covering bio, body of work, photography, and contact. Hand-written React in a single `index.html`, transformed in the browser by Babel Standalone — no build step, no bundler, no CMS. Serves as the *human* counterpart to my product-focused surfaces.

## Status

- **Live and stable.** Hosted on GitHub Pages from `main`.
- **Low-frequency updates.** New project additions, photography refreshes, occasional copy tweaks.
- **No build step**, on purpose. Whole site is one HTML file. Edit and push.

## Stack

- React 18 (UMD) + Babel Standalone (in-browser transform)
- Outfit (sans) + DM Serif Display (display) from Google Fonts
- Custom design system with indigo/cyan/amber palette
- Hosted on GitHub Pages, served from `main` branch root
